# Reunion Compiler

Compiler for the **Reunion** language — a DSL that transpiles `.reu` files into `.plugin` files (Python) for [exteraGram](https://github.com/exteraSquad/exteraGram) and [AyuGram](https://github.com/AyuGram/AyuGram4A) plugins.

---

Documentation, examples, and source code: **[github.com/shareui/reunion-compiler](https://github.com/shareui/reunion-compiler)**
